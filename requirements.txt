# System Requirements
Node.js: v18.0.0 or higher (v20+ recommended)
MongoDB: v6.0 or higher (running locally or remote connection URI)
npm: v9.0.0 or higher

# Installation Steps
1. In the project root (MongoMeals_final):
   npm install
2. In the server directory (MongoMeals_final/server):
   npm install
